DevOps: Automatización y Despliegue

CI/CD y Despliegue Automático

CI/CD (Integración Continua y Despliegue Continuo) es el proceso de subir código constantemente y que este se publique solo. ¡Es una herramienta de trabajo esencial en el mundo real!

Usamos GitHub Actions para configurar una receta (deploy.yml) que le dice a GitHub que publique esta página en GitHub Pages automáticamente cada vez que hago un push. Es el primer paso en DevOps.
